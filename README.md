## Usage

Please install [cc4v](https://github.com/cc4v/cc4v) first.

```bash
# NOTE: You should not use ~/.vmodules for this clone
#       Please `cd ~/Documents` or somewhere else.

$ git clone https://github.com/cc4v/cc4v-examples
$ cd cc4v-examples
$ v run ./hello_world
```
